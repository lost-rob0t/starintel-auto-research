#+title: Starintel Auto Research

* Purpose

This repository is the Emacs-first control plane for Starintel research, design files, SRFCs, implementation promotion, and graph-oriented IDE surfaces.

The Org tree under =roam/= is the source of truth.  Elisp under =lisp/= owns capture, indexing, validation, context bundling, Git workflow helpers, and graph export.

* Repository Shape

#+begin_example
lisp/                         Emacs workflow implementation
roam/design/                  Numbered design files
roam/research/                Research trackers and findings
roam/implement/               Designs promoted for implementation
roam/indexes/                 Roadmaps and project indexes
ui/web/                       Web graph IDE
ui/desktop/                   Desktop graph IDE
#+end_example

* Rule

Do not load the entire research tree into an agent context.  Build a bounded context bundle from the active node, its direct links, and its project index.
