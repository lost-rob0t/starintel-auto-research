# Plan Investigation

**Input:** question, target identifiers, authorization, constraints.

**Output:** one investigation node containing objective, known facts, collection plan, priorities, and stop conditions.

1. State the decision or question the work must support.
2. Separate known facts, assumptions, and unknowns.
3. List the minimum sources needed to close the highest-value gaps.
4. Define collection limits, validation, and completion criteria.

**Stop:** the next collection action is explicit and bounded.

**Script:** optional sibling `run.el`.
